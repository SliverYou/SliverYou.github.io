书单

路漫漫其修远兮，吾将上下而求索。

所列书籍均为已入手并值得推荐的书，将会持续更新。
格式为：[出版时间] + [书名] + [(国籍)作者] + [阅读情况]

Python

基础入门

  • 2016.07 Python编程：从入门到实践 [美] 埃里克·马瑟斯
    • 100% ✔️ [当初入门 Python 的第一本书，可以快速掌握大部分 Python 语法知识和内置数据结构的用法，但是后续的项目描述得不是很具体，有时候只是一笔带过。]
  • 2018.01 Python基础教程（第3版） [挪] Magnus Lie Hetland
    • 100% ✔️ [更加扩展了 Python 的知识体系，弥补了前一本书的不足，代码简洁巧妙，而后续对 Python 应用方面的介绍仿佛打开了 Python 新世界的大门。]

底层原理

  • 2013.07 Learning Python（第5版） [美] Mark Lutz
    • 66% ⏰ [许多人对这本书褒贬不一，觉得 Python 语言简洁巧妙，学习这样厚达1000多页的书，有悖 Python 简单不啰嗦的初衷。而我觉得这本书很详尽的解释了 Python 的内存处理方式和细节特征，讲究从底层解释 Python。虽然有些啰嗦，但确实是本好书。]
  • 2008.06 Python源码剖析 陈儒
    • 0% ❌ [暂时还不需要从源码应用 Python 的角度，选择以后再看。]

奇淫技巧

  • 2017.05 流畅的Python [巴西] Luciano Ramalho
    • 10% ⏰ [看了 Cook Book 后，就拖延了学习这本书的时间了。]
  • 2016.01 Effective Python [美] 布雷特·斯拉特金（Brett Slatkin）
    • 100% ✔️ [可能是因为先看的 Cook Book，对里面介绍的一些技巧觉得不是那么惊艳，但强调的 Pythonic 思想，将会深入我心。]
  • 2015.05 Python Cookbook（第 3 版） [美] David M. Beazley / [美] Brian K. Jones
    • 80% ⏰ [奇书，给出一系列的问题，运用 Python 的方式去解决，学到了很多奇淫技巧。]

计算机视觉

  • 2014.06 Python计算机视觉编程 [瑞典] Jan Erik Solem
    • 15% ⏰ [这本书主要讲述了常用计算机视觉算法的 Python 实现，它更像一本查阅手册，在处理验证码识别时，查阅过本书，得到了启发。]

Web

  • 2014.02 Flask Web开发：基于Python的Web应用开发实战 [美] Miguel Grinberg

  • 2017.05 精通 Django Nigel George
    • 0% ❌ [想找个时间好好学习，并找个项目做一下哈哈。]

安全

    • 5% ⏰ [这本书偏重于应用，其实更多就是介绍库的使用，对我想要的深入学习反而没有太多的价值。]
  • 2015.08 Python黑帽子：黑客与渗透测试编程之道 [美]Justin Seitz
    • 0% ❌ [由于上本书的失望，在安全方面只能先缓一缓了。]

爬虫

  • 2016.03 Python网络数据采集 [美] 米切尔
    • 100% ✔️ [爬虫入门必备，书中代码优美，很有干货，也有具体的实战环节，系统学习后，可以爬去绝大多数的网站了。]
  • 2016.08 用Python写网络爬虫 [澳]理查德 劳森
    • 100% ✔️ [书很薄，代码是 2.X 版本的，我觉得真正的亮点是验证码处理和 Scrapy 框架的使用，可以当成简单的拓展爬虫知识，毕竟书的篇幅很短，不会占用太多时间。]

数据科学

  • 2018.01 Python数据科学手册 Jake VanderPlas
    • 50% ✔️ [等了很久的关于数据科学的书，中文版一出就买了。书中讲解了 IPython 的使用，Numpy、Pandas 和 Matplotlab 也有具体讲解和使用，深入浅出，比之前查阅的国内一些数据科学的书好很多。]
  • 2017.08 Python数据分析基础 [美] Clinton W. Brownley
    • 100% ⏰ [虫子书，更多的是 Python 处理数据，对大量数据可以编写脚本，进行可视化操作，方便好用。]
  • 2017.11 Python数据科学入门 [俄] Dmitry Zinoviev
    • 100% ✔️ [这本书很薄，但内容很丰富，像是数据科学的地图，不完全指北。]
