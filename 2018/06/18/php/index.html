PHP 初探

PHP 在 Web 世界中的地位

PHP 是一门编程语言，主要用于构建网站。PHP 程序一般不是运行在桌面电脑中供一个电脑使用的，而是运行在 Web 服务器中，由许多人在他们各自的电脑中使用 Web 浏览器访问。Web 服务器使用 PHP 创建响应或文档，然后发回浏览器。

PHP 的优势

  1. PHP 是免费的：使用 PHP 无需给任何人付钱，可以将它运行在破旧不堪的电脑上，也可以在企业级项目中使用 PHP，无需付其他费用。
  2. PHP 是自由的：PHP是开源项目，在任何时候都可以查看它的源码，并且可以深入 PHP 引擎，一探究竟。
  3. PHP 跨平台：只要有 Web 服务器，PHP 可以运行在 Windows，Mac OS X，Linux 和多数 UNIX 版本中使用。
  4. PHP 使用广泛：全世界有超过两亿个网站使用 PHP，这包括了数不清的小型网站和大型网站。
  5. PHP 隐藏了复杂度：PHP 有许多有用的功能，如缓存、动态生成图像等，让用户能更加关心输入输出，而不是复杂的功能实现。
  6. PHP 为了 Web 编程而生：PHP 一开始就是为了生成网页而生的，这意味着常见的 Web 任务通过 PHP 都能简单快速的实现。

PHP 程序的基本原则

起始和结束标签

PHP 可以使用 <?pho 作为起始标签，?> 作为结束标签。PHP 引擎会忽略这两个标签之外的任何内容。起始标签之前的文本，或者结束标签之后的文本都不经 PHP 引擎的处理，直接打印出来。并且，一个 PHP 文件中可以有多对起始和结束标签。

空白

PHP 程序一行代码中可以编写多个 PHP 语句，只要相互之间使用分号隔开即可。语句之间可以有任意多个空行，但最好要适度空行，提高代码可读性。

大小写

在 PHP 中，语言关键字（如 print）和函数名是不区分大小写的。在程序中编写关键字和函数名时，PH 引擎不介意你使用大写字母、小写字母或混用。

比较

比较浮点数

因为在计算机中，数字都是以二进制存储的，所以浮点数在计算机内部存储的值可能与赋予的值稍微不同，例如50.0可能会存储为50.00000000002。想要比较两个浮点数是否相等，不能使用 ==，而要检查两数之差是否小于可接受的阈值。
